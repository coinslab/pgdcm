# Push Q-Matrix to Cytoscape

Push Q-Matrix to Cytoscape

## Usage

``` r
QMatrix2CytoNodes(Q, compute = "dina", title = "Scenario2_QMatrix")
```

## Arguments

- Q:

  A data.frame Q-Matrix setup.

- compute:

  Character. Default compute rule.

- title:

  Character. The Network Title in Cytoscape.

## Value

SUID of pushed Cytoscape Network.
