# Add Edge

Add Edge

## Usage

``` r
edgeadd(theigraph, fromnodechoice, tonodechoice, colornickname)
```

## Arguments

- theigraph:

  The igraph object

- fromnodechoice:

  The source node

- tonodechoice:

  The target node

- colornickname:

  The color nickname

## Value

The updated igraph object
