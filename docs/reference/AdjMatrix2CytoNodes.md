# Push Adjacency Matrix to Cytoscape

Push Adjacency Matrix to Cytoscape

## Usage

``` r
AdjMatrix2CytoNodes(Adj, compute = "dina", title = "Scenario3_AdjMatrix")
```

## Arguments

- Adj:

  A data.frame Adjacency Matrix setup.

- compute:

  Character. Default compute rule.

- title:

  Character. The Network Title in Cytoscape.

## Value

SUID of pushed Cytoscape Network.
