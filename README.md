# PGDCM

Probabilistic Graphical Diagnostic Classification Model (development)

The main program is "runPGDCM.R"
Specify graph structure using the prefix of the nodes file and the edges file (e.g., DTMRedges.csv and DTMRnodes.csv)
Specify data set as an EXCEL spreadsheet using a format such as: DTMRdata.xlsx
The graph structure and data set have to be compatible and the program tries to check this

Tutorial papers can be found in the folder "TUTORIALPAPERS"
